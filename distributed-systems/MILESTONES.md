# Distributed Systems — Hazelcast

## Project: Distributed Cache & Compute Layer

Build a caching and distributed compute layer for a data service. Understand clustering, data structures, and distributed execution.

## Milestones

1. **Cluster & Data Structures** — Start a cluster, use distributed maps, queues, sets.
2. **Distributed Computing** — EntryProcessors, executor service, running compute on data.
3. **Serialization & Performance** — Compact serialization, near-cache, benchmarking.
4. **Persistence & Resilience** — Persistence, split-brain, backup counts.
5. **Jet Stream Processing** — Hazelcast Jet pipelines for streaming aggregation.
