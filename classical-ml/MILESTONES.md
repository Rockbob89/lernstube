# Classical ML — scikit-learn

## Project: Airbnb Price Predictor

Train a model that predicts listing price from features like location, room type, reviews, and availability.

## Milestones

1. **Estimator API & Preprocessing** — Train/test split, scaling, encoding categoricals. Fit a baseline model.
2. **Linear Models** — Linear/Ridge regression. Evaluate with RMSE, R². Feature importance.
3. **Tree-Based Models** — Decision tree, random forest, gradient boosting. Compare to linear.
4. **Tuning & Pipelines** — GridSearch/RandomSearch, sklearn Pipelines, cross-validation.
5. **Evaluation & Reporting** — Learning curves, residual analysis, final model selection.
