# Python — Task List

1. **Args & Kwargs** — Function signatures, unpacking, forwarding
2. **Data Structures Deep Dive** — Sets, tuples, namedtuples, defaultdict, Counter
3. **Comprehensions & Generators** — List/dict/set comprehensions, generator expressions, `yield`
4. **Closures & `nonlocal`** — Inner functions capturing enclosing scope, mutable state
5. **Basic Decorators** — Writing a simple wrapper, `functools.wraps`
6. **Decorators with Arguments** — Decorator factories, three-level nesting
7. **Forwarding Arguments** — `*args, **kwargs` in decorator wrappers, inspecting kwargs
8. **Context Managers** — `with`, writing your own (`__enter__`/`__exit__`, `contextlib`)
9. **Error Handling** — Custom exceptions, exception chaining, `try/except/else/finally`
10. **File I/O & Serialization** — Reading/writing files, JSON, CSV, `pathlib`
11. **Type Hints & Tooling** — Annotations, `mypy`, `typing` module, dataclasses
12. **Modules & Packages** — Project structure, `__init__.py`, relative imports, entry points
13. **Testing** — `pytest`, fixtures, parametrize, mocking basics
14. **Virtual Environments & Dependency Management** — venv, pip, `pyproject.toml`, uv
15. **Capstone: CLI Tool** — Build a CLI app that ties together everything above
