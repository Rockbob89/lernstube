# Task 06: Capstone — Dashboard

## Objective
Create a multi-panel static figure from a real dataset, exported as PNG and PDF. Combine everything from tasks 1-5.

## Requirements
- Use a real dataset (download one or use a bundled one like iris, tips, etc. via seaborn/sklearn)
- Minimum 4 panels in a non-trivial layout (not just a 2x2 grid)
- Must include: at least one line/scatter, one bar/histogram, one heatmap or custom-styled plot
- Apply a consistent style/theme across all panels
- Add a figure-level title (`fig.suptitle`)
- Export to both `dashboard.png` (dpi=200) and `dashboard.pdf`
- Code in a single `dashboard.py` script

## Deliverables
- `dashboard.py`
- `dashboard.png`
- `dashboard.pdf`
