# NumPy — Task List

1. **Arrays & dtypes** — Creating arrays, shapes, dtypes, reshaping
2. **Indexing & Slicing** — Fancy indexing, boolean masks, views vs copies
3. **Vectorized Operations** — Broadcasting, ufuncs, avoiding loops
4. **Linear Algebra** — Dot products, matrix ops, `np.linalg`
5. **Aggregation & Statistics** — sum, mean, std along axes, argmin/argmax
6. **Random & Sampling** — `np.random`, seeds, distributions, shuffling
7. **Capstone: Image Manipulator** — Load an image as array, apply transformations (rotate, blur, threshold) using only NumPy
