# Streaming & OLAP — Flink + Druid

## Project: Real-Time Event Analytics

Build a streaming pipeline that ingests events, processes them with Flink, and serves analytics queries from Druid.

## Milestones

1. **Flink Fundamentals** — DataStream API, sources, sinks, local execution.
2. **Windowing & State** — Time windows, aggregations, stateful processing, checkpoints.
3. **Druid Fundamentals** — Architecture, ingestion specs, segments, local cluster.
4. **Flink → Druid** — Stream processed data into Druid. Query with Druid SQL.
5. **Analytics Queries** — Time-series aggregations, TopN, groupBy. Build a query layer.
