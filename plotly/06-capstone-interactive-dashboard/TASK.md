# Task 06: Capstone — Interactive Dashboard

## Objective
Build a Dash app with multiple linked charts from a real dataset. Combine everything from tasks 1-5.

## Requirements
- Use a real dataset (download or use a bundled one)
- Dash app with at least 3 interactive charts
- At least 2 filter controls (dropdowns, sliders, or radio buttons)
- Charts must be linked: filtering in one control updates multiple charts
- Include at least one map or 3D plot
- Use Graph Objects for at least one chart (not just Plotly Express)
- Clean layout with sections/headings
- Code in a single `app.py`

## Deliverables
- `app.py`
- Brief `NOTES.md` documenting: dataset used, what each chart shows, design decisions
