# Visualization — Matplotlib + Plotly

## Project: Berlin Airbnb Visual Report

Build static publication-quality charts (Matplotlib) and an interactive exploration tool (Plotly) from the cleaned Airbnb data you produced in Data Wrangling.

## Milestones

1. **Static Charts** — Bar, histogram, and scatter plots with Matplotlib. Axes, labels, styling.
2. **Multi-Panel Layouts** — Subplots, figure composition, annotations. One polished multi-chart figure.
3. **Interactive Plots** — Rebuild key charts in Plotly. Hover, zoom, filtering.
4. **Dashboard** — Combine Plotly charts into a Dash mini-app or standalone HTML report.
