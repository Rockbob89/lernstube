# IaC — Pulumi

## Project: Deploy a Data Service to Cloud

Use Pulumi (Python) to deploy one of your earlier projects — infrastructure as real code, not YAML.

## Milestones

1. **Pulumi vs Terraform** — Concepts, state, providers, stacks. First `pulumi up`.
2. **Resources & Providers** — Create cloud resources (storage, compute, networking).
3. **Inputs, Outputs, Config** — Pulumi's apply model, stack config, secrets.
4. **Component Resources** — Reusable abstractions, packaging infrastructure as components.
5. **Testing & CI** — Unit tests for infrastructure, integration into a pipeline.
