# LangChain — Task List

1. **Models & Prompts** — LLM/Chat model wrappers, prompt templates, output parsers
2. **Chains** — Sequential chains, LCEL (LangChain Expression Language), RunnablePassthrough
3. **Retrieval & RAG** — Document loaders, text splitters, embeddings, vector stores
4. **Memory & State** — Conversation memory types, message history
5. **Agents & Tools** — Tool definitions, agent types, ReAct pattern
6. **Evaluation** — LangSmith tracing, evaluation datasets, scoring
7. **Capstone: RAG Application** — Build a retrieval-augmented generation app over a custom document corpus with a FastAPI backend
