# Deep Learning — PyTorch + TensorFlow

## Project: Image Classifier

Build an image classifier (e.g., room type from listing photos). Implement in PyTorch first, then rebuild in TensorFlow/Keras to compare.

## Milestones

1. **Tensors & Autograd** — NumPy-to-tensor bridge, automatic differentiation in PyTorch.
2. **First Neural Net (PyTorch)** — nn.Module, training loop, loss, optimizer. Classify on a simple dataset.
3. **CNNs & Transfer Learning (PyTorch)** — Convolutions, pretrained models, fine-tuning.
4. **TensorFlow/Keras Rebuild** — Same model in Keras. Compare APIs, training, and ecosystem.
5. **Saving & Serving** — Export models, basic inference pipeline.
