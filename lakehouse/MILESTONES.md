# Lakehouse — Databricks + Data Lakehouse Concepts

## Project: Medallion Architecture

Build a bronze/silver/gold lakehouse pipeline. Understand storage formats, schema evolution, and how Databricks fits in.

## Milestones

1. **Lake vs Warehouse vs Lakehouse** — Concepts, trade-offs, when to use what.
2. **Storage Formats** — Parquet, Delta Lake, Iceberg. Hands-on with local files.
3. **Medallion Pipeline** — Bronze (raw), Silver (clean), Gold (aggregates). Build it locally with Delta/Spark.
4. **Databricks Platform** — Notebooks, clusters, Unity Catalog, Jobs. Port the pipeline.
5. **SQL Warehouses & Serving** — Query the gold layer, build a simple dashboard or API on top.
