# Data Wrangling — NumPy + Pandas

## Project: Analyze Berlin Airbnb Listings

You'll work with the Inside Airbnb dataset for Berlin — real, messy, public data. By the end you'll have a clean analysis pipeline that answers concrete questions about pricing, availability, and neighborhoods.

Dataset: http://insideairbnb.com/get-the-data/ → Berlin → `listings.csv` (detailed)

## Milestones

1. **Load and Explore** — Read the CSV, inspect shape/dtypes/nulls, select columns with Pandas. Use NumPy to understand what's underneath.
2. **Clean and Transform** — Handle missing values, fix dtypes, parse prices, engineer features. NumPy vectorized ops where they make sense.
3. **Group, Aggregate, Answer** — groupby, pivot tables, window operations. Answer: which neighborhoods are cheapest? How does price vary by room type?
4. **Export and Validate** — Write clean output, sanity-check with NumPy stats, produce a summary report as CSV/JSON.
