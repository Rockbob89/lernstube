# LLM Apps — LangChain

## Project: RAG Over Your Own Docs

Build a retrieval-augmented generation app that answers questions over a local document corpus.

## Milestones

1. **Models & Prompts** — LLM wrappers, prompt templates, output parsing.
2. **Chains & Retrieval** — Document loaders, embeddings, vector stores, retrieval chains.
3. **RAG Pipeline** — End-to-end question answering with source attribution.
4. **Agents & Tools** — Give the LLM tools to call. Build a simple agent loop.
5. **Evaluation** — Test retrieval quality and answer accuracy.
