# Web APIs — FastAPI + Flask

## Project: Airbnb Listings API

Build a REST API serving the cleaned Airbnb data. Start with FastAPI, then port the core to Flask to compare frameworks.

## Milestones

1. **FastAPI Basics** — Routes, path/query params, response models with Pydantic.
2. **Database & CRUD** — Persist listings, wire up SQLAlchemy or similar.
3. **Auth & Middleware** — Token auth, CORS, error handling, request logging.
4. **Flask Port** — Rebuild the read endpoints in Flask. Compare routing, validation, async.
5. **Testing** — Test both implementations with pytest.
