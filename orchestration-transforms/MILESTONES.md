# Orchestration & Transforms — Airflow + dbt

## Project: Automated ELT Pipeline

Build a scheduled pipeline that ingests raw data, transforms it with dbt SQL models, and orchestrates the whole flow with Airflow.

## Milestones

1. **Airflow Core** — DAGs, tasks, operators, scheduling. Run a simple pipeline locally.
2. **dbt Project** — Models, sources, refs. Transform raw data into clean tables.
3. **dbt Testing & Docs** — Schema tests, custom tests, generated documentation.
4. **Integration** — Trigger dbt from Airflow. Sensors, task dependencies, error handling.
5. **Production Patterns** — Incremental models, backfills, alerting.
