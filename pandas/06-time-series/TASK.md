# Task 06: Time Series

## Objective
Work with datetime indices, resampling, and rolling window operations.

## What to Learn
- `pd.to_datetime()`, `pd.Timestamp`, `pd.DatetimeIndex`
- `.dt` accessor: year, month, dayofweek, etc. ŌĆö like `.str` for strings, but for datetime columns
  ```python
  s = pd.Series(pd.to_datetime(["2024-01-15", "2024-03-22"]))
  s.dt.month       # Series([1, 3], dtype: int32)
  s.dt.dayofweek   # 0=Monday ... 6=Sunday
  ```
- `resample()`: downsampling and upsampling ŌĆö requires a DatetimeIndex; works like `groupby` but over time intervals
  ```python
  # daily data ŌåÆ weekly sum
  df.resample("W")["value"].sum()
  # hourly data ŌåÆ daily OHLC
  df.resample("D")["price"].ohlc()
  ```
- `rolling()`, `expanding()`, `ewm()` ŌĆö `rolling(n)` uses a fixed window of n periods; `expanding()` grows from start to current row; `ewm` applies exponential weights
  ```python
  s.rolling(window=3).mean()    # NaN for first 2 rows, then 3-period average
  s.expanding().mean()          # cumulative mean from row 0
  s.ewm(span=3).mean()          # exponentially weighted mean
  ```
- Timezone handling: `tz_localize`, `tz_convert` ŌĆö `tz_localize` assigns a timezone to naive datetimes; `tz_convert` changes from one timezone to another
  ```python
  s = pd.DatetimeIndex(["2024-01-01 12:00"])
  s = s.tz_localize("UTC")          # naive ŌåÆ UTC
  s.tz_convert("US/Eastern")        # UTC ŌåÆ Eastern
  ```

## Exercises

### 1. `create_date_range_df(start, end, freq, value_func) -> pd.DataFrame`
Create a DataFrame with DatetimeIndex from start to end at given freq. Column `value` is generated by `value_func(index)`.
```python
create_date_range_df("2024-01-01", "2024-01-05", "D", lambda idx: range(len(idx)))
#             value
# 2024-01-01      0
# 2024-01-02      1
# ...
```

### 2. `resample_ohlc(df, rule) -> pd.DataFrame`
Given a DataFrame with DatetimeIndex and a `price` column, resample to `rule` frequency and compute OHLC (open, high, low, close).
```python
# Hourly data resampled to daily
resample_ohlc(hourly_df, "D")
#             open  high  low  close
# 2024-01-01  ...   ...  ...    ...
```

### 3. `rolling_zscore(series, window) -> pd.Series`
Compute rolling z-score: (value - rolling_mean) / rolling_std.
```python
s = pd.Series([1, 2, 3, 10, 3, 2, 1], index=pd.date_range("2024-01-01", periods=7))
rolling_zscore(s, 3)
# First (window-1) values are NaN, then z-scores
```

### 4. `business_day_stats(df, tz) -> dict`
Given a DataFrame with DatetimeIndex (UTC), convert to `tz`, then return dict with: `busiest_day` (day name with most rows), `weekend_pct` (% of rows on weekends), `hour_mode` (most common hour).
```python
business_day_stats(df, "US/Eastern")
# {'busiest_day': 'Wednesday', 'weekend_pct': 12.5, 'hour_mode': 14}
```
