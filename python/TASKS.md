# Python — Task List

1. **Args & Kwargs** — Function signatures, unpacking, forwarding
2. **Data Structures Deep Dive** — Sets, tuples, namedtuples, defaultdict, Counter
3. **Comprehensions & Generators** — List/dict/set comprehensions, generator expressions, `yield`
4. **Decorators & Closures** — Writing and applying decorators, closures, `functools.wraps`
5. **Context Managers** — `with`, writing your own (`__enter__`/`__exit__`, `contextlib`)
6. **Error Handling** — Custom exceptions, exception chaining, `try/except/else/finally`
7. **File I/O & Serialization** — Reading/writing files, JSON, CSV, `pathlib`
8. **Type Hints & Tooling** — Annotations, `mypy`, `typing` module, dataclasses
9. **Modules & Packages** — Project structure, `__init__.py`, relative imports, entry points
10. **Testing** — `pytest`, fixtures, parametrize, mocking basics
11. **Virtual Environments & Dependency Management** — venv, pip, `pyproject.toml`, uv
12. **Capstone: CLI Tool** — Build a CLI app that ties together everything above
